    doormat

  2. How about something a little different for navigating through your site content?

  3. How to create a doormat

    It's actually real simple.

    1. Include doormat.js and doormat.css in your page.
    2. Create the DOM structure you want to use. This needs to be a containing element of children. All of the children will be used as panels. I recommend something like an ol or ul.
    3. Invoke the doormat function passing in the container element as a parameter. For example; 
      var container = document.querySelector('ol');
      var myDoormat = new doormat(container);

  4. That's it really.
